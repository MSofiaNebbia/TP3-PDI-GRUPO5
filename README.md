## Descripción

### Problema 1: Detección automática de dados en video
Se desarrolló un algoritmo que procesa automáticamente videos de tiradas de 5 dados rojos sobre fondo verde. El sistema detecta los períodos en los que los dados están quietos, identifica cada dado y cuenta sus puntos, valida que la cantidad y los valores sean correctos, y genera un video de salida con los bounding boxes y etiquetas anotadas sobre cada dado.
